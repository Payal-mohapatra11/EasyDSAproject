# EasyDSA 
### Visual Data Structures & Algorithms Learning Platform

EasyDSA is an interactive web-based platform designed to help students understand **Data Structures and Algorithms (DSA)** through **visualizations and practical demonstrations** instead of only theory.  
The project focuses on making DSA **simple, engaging, and accessible**, especially for **beginners and deaf students**.

---

## Key Features

-  **Visual DSA Learning**
  - Interactive visualizers for arrays, sorting, and searching algorithms
  - Step-by-step animations for better understanding

- **User Authentication**
  - Secure Signup and Login system
  - User-based access to learning resources

- **AI Chatbot Support**
  - Chatbot integrated to help students clear doubts instantly
  - Provides explanations related to DSA topics

- **Responsive & Clean UI**
  - Built with Tailwind CSS for modern and responsive design
  - Beginner-friendly interface

## Project Objective

- To make DSA learning **visual and intuitive**
- To help students understand **how algorithms work internally**
- To support inclusive learning for **deaf and beginner students**
- To reduce fear of DSA through interactive learning
